# iopenclawwx

```bash
openclaw plugins install iopenclawwx
cd ~/.openclaw/extensions/iopenclawwx
npm run config-init  # 输入你的 API Key
openclaw gateway restart
```
目前小程序正在审核中，通过后应该就可以使用了 这是比较简单的接入到微信的方式
